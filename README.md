## 🚀 Featured Projects

### 🔬 [Adversarial Privacy Mask](https://github.com/rehanali1092/adversarial-privacy-filter) `Ongoing`
Custom loss function integrated with PGD attacks that empirically outperforms traditional FGSM and PGD on facial recognition models.  
**Tech:** Python, PyTorch, Adversarial ML, CASIA-WebFace

### 🌳 [Date Palm Tree Detection System](https://github.com/MuhammadYasir85a/PalmTreeDetection)
End-to-end computer vision system using YOLOv11 for detecting and counting date palm trees from aerial imagery, with BoT-SORT tracking, Flask API, and React Native frontend.  
**Tech:** Python, YOLOv11, BoT-SORT, OpenCV, Flask, React Native, Roboflow

### ☁️ [Distributed Student Issue Management System](https://github.com/MuhammadYasir85a/DSIMS-distributed-student-issue-management) `Ongoing`
Distributed web application with MongoDB sharding, deployed on Microsoft Azure with JWT authentication and Role-Based Access Control.  
**Tech:** Node.js, Express, MongoDB, Microsoft Azure, JWT

### 🛡️ [E-Commerce Enterprise Network](https://github.com/MuhammadYasir85a/E-Commerce-Company-with-Load-Balanced-Web-Servers-AAA)
Secure enterprise network with DNS load balancing, AAA security, VLAN segmentation, and Layer-2 protections.  
**Tech:** Cisco Packet Tracer, Cisco IOS, Network Security

### 🌦️ [Probabilistic Weather Prediction System](https://github.com/MuhammadYasir85a/Probabilistic-Weather-Prediction)
Statistical machine learning model forecasting precipitation events using Naive Bayes classification on atmospheric variables.  
**Tech:** Python, Naive Bayes, NumPy, Pandas, SciPy
