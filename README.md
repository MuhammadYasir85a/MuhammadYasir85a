### Adversarial Privacy Mask
**Status:** Ongoing Research

A team project where I am co-developing a custom loss function integrated with PGD attacks for adversarial perturbation generation. The approach empirically outperforms standard FGSM and PGD baselines on facial recognition models. Currently scaling validation from LFW to CASIA-WebFace using PyTorch.

**Technologies:** Python, PyTorch, Adversarial ML, CASIA-WebFace

[View Repository](https://github.com/rehanali1092/adversarial-privacy-filter)

<td width="50%" valign="top">

### Date Palm Tree Detection System
**Status:** Completed

End-to-end computer vision system that detects and counts date palm trees from aerial imagery using YOLOv11 deep learning model. Integrated BoT-SORT multi-object tracking for video frame consistency. Built Flask REST API with React Native mobile frontend.

**Technologies:** Python, YOLOv11, BoT-SORT, OpenCV, Flask, React Native

[View Repository](https://github.com/MuhammadYasir85a/PalmTreeDetection)

**Technologies:** Node.js, Express, MongoDB, Microsoft Azure, JWT

**Technologies:** Cisco Packet Tracer, Cisco IOS, Network Security, AAA
